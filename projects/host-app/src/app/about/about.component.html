Sistema de Cadastro de Parceiros e Empresas Externa.

Projeto desenvolvido com Angular na versão 15.

Aplicação dividida em partes com a implementação de Micro-Front-ends.

Para o uso de Micro-Front-ends é utilizado o Module Federation

Implementação e uso com Docker

Uso de configurações de servidor com Nginx

Implementação de Testes Unitário/Integração via Karma

Implementação de Testes Automatizado via Cypress

Implementação com GitHub Pages

Uso de componentes com Angular Material

Uso de componentes com o Bootstrap
